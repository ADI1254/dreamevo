## 🔧 Audio File Optimization

### Before Using Files

1. **Trim Silence**
   ```bash
   # Using FFmpeg (free)
   ffmpeg -i input.mp3 -af silenceremove=1:0.1:0.01 output.mp3
   ```

2. **Normalize Volume**
   ```bash
   # Normalize to -16 LUFS
   ffmpeg -i input.mp3 -af loudnorm=I=-16:TP=-1.5:LRA=11 output.mp3
   ```

3. **Compress for Web**
   ```bash
   # 192kbps MP3
   ffmpeg -i input.mp3 -q:a 3 -b:a 192k output.mp3
   ```

### File Size Reference

After optimization:
- Orchestral Theme (2-4 min): 3-6 MB → 2-4 MB optimized
- Ambient Pad (1 min): 1-2 MB → 500 KB optimized
- SFX (10-30 sec): 300-500 KB → 100-200 KB optimized
