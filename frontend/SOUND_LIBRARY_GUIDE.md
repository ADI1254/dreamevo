### Step 1: Know What You Need
```
Time: 0:02
Emotion: Urgency, dread
Story: Message arrives
Needed: Low, ominous tone that feels SCARY but not cartoonish
```

### Step 2: Search Smart
```
DON'T Search: "scary sound"  (too generic)
DO Search: "dark drone", "ominous bass", "dread tone"

DON'T Search: "water"  (thousands of results)
DO Search: "water wade loop", "river splash", "chest-high water"
```

### Step 3: Preview & Evaluate
```
Listen for:
  ✅ Professional quality (no hum, noise)
  ✅ Fits emotional tone (not overacting)
  ✅ Correct duration (~2-3s most)
  ✅ Doesn't have music (pure SFX)
  ✅ Stereo or mono OK (mono is fine)
```

### Step 4: Download & Test
```
Import to DAW
Hear it in context with narration
Does it ENHANCE or DISTRACT?
If enhance: Keep
If distract: Delete and try another
```

## 📝 SOUND SELECTION PHILOSOPHY

### For Movie-Level Quality:

1. **Authenticity > Drama**
   - Real footsteps on leaves sound better than synthetics
   - Actual water splashes > exaggerated whooshes
   - Honest recordings > overly processed

2. **Subtlety > Obviousness**
   - -12dB bell chime > loud clanging
   - Implied danger > dramatic stabs
   - Suggestion > explanation

3. **Layering > Single Sounds**
   - Combine 2-3 elements at key moments
   - Example: Low drone + wind + distant bird = complete moment
   - But never more than 3 layers (muddy)

4. **Space Awareness**
   - All sounds should feel like they're in the same acoustic space
   - If narration has reverb, SFX should too
   - Match decay times (not 0.3s reverb + 2s reverb)

5. **Emotional Truth**
   - Sound choice should serve story, not ego
   - Would THIS sound exist in a real jungle?
   - Does it match the mood (calm, not aggressive)?

## ⭐ PRO TIPS

### Quality Indicators in Metadata:
- Look for: "48kHz" or "96kHz" (not 44.1kHz)
- Look for: "Stereo" (wider immersion)
- Look for: "Unprocessed" or "raw" (you add reverb)
- Avoid: "Gated reverb", "heavily compressed", "lo-fi"

### Red Flags:
- ❌ File size too small (indicates lossy)
- ❌ Weird artifacts in waveform
- ❌ Doesn't preview in library player
- ❌ Comments say "too fake" or "generic"

### Green Flags:
- ✅ 4.8+ stars rating
- ✅ 100+ downloads
- ✅ Comments praise realism
- ✅ Clean waveform, no noise floor
