# World Card Images

Place your world card background images in this directory:

- `journey-card.jpg` - Journey world card (desert, mountain, figure walking)
- `sanctuary-card.jpg` - Sanctuary world card (tree, waterfall, glowing orbs)
- `exploration-card.jpg` - Exploration world card (blimp, crystals, cave entrance)

## Image Requirements:
- Format: JPG or PNG
- Recommended size: 600x800px or larger (vertical orientation)
- Aspect ratio: 3:4 or similar
- The images will be automatically cropped and scaled to fit the cards

## Fallback:
If images are not found, the cards will use beautiful gradient backgrounds as fallback.
