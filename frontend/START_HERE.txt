⚡ VOLUME DUCKING (Smart Audio Management):

    WHEN NARRATION STARTS:
    ┌──────────────────────────────────────────────────────────────┐
    │ Orchestra: 40% ──────► 15% (DUCKS for voice)                │
    │ SFX:       Auto adjust downward                             │
    │ Voice:     ENTERS at 95% (CRYSTAL CLEAR!)                  │
    │ Ambient:   Continues at 20%                                │
    └──────────────────────────────────────────────────────────────┘

    WHEN NARRATION ENDS:
    ┌──────────────────────────────────────────────────────────────┐
    │ Orchestra: 15% ──────► 40% (RESTORES smoothly)             │
    │ SFX:       Return to normal                                 │
    │ Voice:     Silent                                           │
    │ Ambient:   Continues                                        │
    └──────────────────────────────────────────────────────────────┘

    Result: Voice always sits perfectly on top of music!

🐛 TROUBLESHOOTING:

    Issue: Audio won't play
    Solution: Check browser console (F12) for errors
             Verify audio file paths are correct
             Check browser autoplay policy

    Issue: Narration sounds robotic
    Solution: Use ElevenLabs API for premium voice
             Get free API key from elevenlabs.io
             Pass to: exp.play('api-key')

    Issue: Scene changes at wrong time
    Solution: Edit sceneMarkers in viking_video_experience.js
             Adjust start/end times to match YOUR video

    Issue: Music ducks too much/too little
    Solution: Adjust orchVolumeDucked in viking_audio_system.js
             Try: 0.10 for more ducking
             Try: 0.25 for less ducking
